Kebijakan Privasi

1. Pendahuluan

Vaultify adalah aplikasi penyimpanan aman yang dirancang untuk melindungi file dan catatan pribadi Anda menggunakan enkripsi. Privasi Anda adalah prioritas utama kami.

2. Pengumpulan Data

Vaultify tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi Anda ke server eksternal mana pun. Semua data sepenuhnya berada di perangkat Anda.

3. Penyimpanan Data

Semua file, catatan, dan informasi disimpan secara lokal di perangkat Anda dalam keadaan terenkripsi. Vaultify tidak menggunakan penyimpanan cloud atau sinkronisasi eksternal.

4. Tanggung Jawab Pengguna

Pengguna sepenuhnya bertanggung jawab atas keamanan dan pengelolaan data mereka sendiri. Kehilangan master password akan mengakibatkan kehilangan akses permanen terhadap data yang terenkripsi. Vaultify tidak dapat memulihkan atau mereset password Anda.

5. Keamanan

Vaultify menggunakan teknik enkripsi modern untuk melindungi data Anda. Namun, keamanan juga bergantung pada pengguna dalam menjaga kerahasiaan password dan perangkat mereka.

6. Akses Pihak Ketiga

Vaultify tidak membagikan data Anda kepada pihak ketiga mana pun karena aplikasi ini tidak memiliki akses ke data Anda.

7. Perubahan Kebijakan

Kebijakan privasi ini dapat diperbarui dari waktu ke waktu. Perubahan akan ditampilkan pada halaman ini.

1. Deskripsi Layanan

Vaultify adalah aplikasi penyimpanan digital yang memungkinkan pengguna menyimpan file dan catatan secara aman menggunakan enkripsi lokal di perangkat pengguna. Aplikasi ini tidak menggunakan server eksternal, tidak menyediakan penyimpanan cloud, dan tidak melakukan sinkronisasi data.

2. Kepemilikan dan Kontrol Data

Seluruh data yang disimpan dalam Vaultify sepenuhnya berada di bawah kendali pengguna. Vaultify tidak memiliki akses, tidak menyimpan salinan, dan tidak dapat membaca atau memulihkan data pengguna.

3. Keamanan dan Enkripsi

Vaultify menggunakan metode enkripsi untuk melindungi data. Namun, pengguna memahami bahwa tidak ada sistem yang sepenuhnya aman. Pengguna bertanggung jawab menjaga keamanan perangkat dan kerahasiaan password.

4. Tanggung Jawab Pengguna

Pengguna bertanggung jawab penuh atas penggunaan aplikasi, termasuk menjaga password dan data. Kehilangan password akan mengakibatkan data tidak dapat diakses kembali.

5. Tidak Ada Pemulihan Data

Vaultify tidak menyediakan fitur pemulihan password atau data. Semua risiko kehilangan data sepenuhnya menjadi tanggung jawab pengguna.

6. Batasan Tanggung Jawab

Vaultify tidak bertanggung jawab atas kehilangan data, kerusakan perangkat, atau akses tidak sah yang terjadi akibat kelalaian pengguna atau faktor di luar kendali aplikasi.
